# Mythic Rift

لعبة معارك بطاقات أصلية تعمل مباشرة في المتصفح. التوزيع المرئي للشاشة والساحة وشريط القتال مقاس من فيديو مرجعي بدقة 386×848.

## التشغيل

```powershell
cd "C:\Users\asumm\OneDrive\سطح المكتب\تقارير برندة\game play"
npm start
```

ثم افتح: http://localhost:3000

محرك القتال الثلاثي الجديد: http://localhost:3000/battle3d.html

إذا كان المنفذ 3000 مشغولًا واللعبة مفتوحة بالفعل، يكفي تحديث الصفحة.

## الشاشات

- الشاشة الرئيسية: موارد، ملف اللاعب، القلعة، معاينة الرزمة، وزر المعركة.
- المعركة: ساحة بمسارين وجسرين، ثلاثة أبراج لكل فريق، يد من أربع بطاقات، طاقة، مقذوفات، استهداف، وقت إضافي.
- مجموعة الوحوش: رزمة من ثماني بطاقات، شبكة البطاقات، مستويات، تكلفة، ونافذة تفاصيل بمعاينة حركة.

## القيم المثبتة

- عالم منطقي 36,000×64,000، وشبكة 36×64، وكل خلية 1,000 وحدة.
- `data/tilemap.csv` يُستخدم فعليًا في البحث عن المسار.
- محاكاة ثابتة 20Hz، وطاقة قصوى 10، وبداية 6.
- زمن أساسي 180 ثانية ووقت إضافي 120 ثانية.
- مراحل حركة الوحش: نشر، مشي، اكتساب هدف، تحضير ضربة، ضرب/إطلاق، استشفاء، وسقوط.

## الهوية

الاسم، الهوية، أسماء الوحوش، ورسوماتها أصلية ومولدة لهذا المشروع. لا يُستخدم شعار اللعبة المرجعية أو شخصياتها أو كودها.
