# BARNDAKSA_CAFE_CINEMATIC_ANIMATIONS_PATCH

باتش يضيف أنيميشن فاخر وخفيف لصفحة الفرع الإلكتروني، يشمل:

- كروت المنتجات.
- كروت العروض والخصومات.
- كروت وخيارات الحجوزات.
- الهيرو والانتقالات داخل صفحات الفرع.
- تأثيرات دخول متدرجة، hover ثلاثي الأبعاد، shine، وstagger animation.

## التشغيل

انسخ مجلد `BARNDAKSA_CAFE_CINEMATIC_ANIMATIONS_PATCH` داخل جذر المشروع ثم شغل:

```bash
node BARNDAKSA_CAFE_CINEMATIC_ANIMATIONS_PATCH/apply-patch.cjs
npm exec tsc -- --noEmit
npm run build
```

لا يوجد تعديل على قاعدة البيانات.
